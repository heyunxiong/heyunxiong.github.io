SCG 静态内容生成器

What is a Static Site Generator

  • Docusaurus
    来自Facebook的开源内容生成器
    技术帖子是由它生成的，我还尝试了其他类似的文档生成器，例如“vuepress、gitbook”，但我选择它是因为它的用法和用户界面更友好

  • Hexo
    Hexo 是我的第一个 SCG，我非常喜欢它。 它易于使用，并具有精彩的主题和插件/功能，可以帮助自定义您的博客
    现在我用的是我自己调整的简洁大方的Keep主题来记录我的思考和日常发帖
    我尝试过主题Volantis，我会给它5颗星； Next 是我之前选择的一个，很棒很简单的主题和风格

  • Hugo
    最快的SCG。 如果您有很多博客文章，您应该尝试一下
    该主页使用的是 lynx 主题
    和由Eternity生成的画廊
