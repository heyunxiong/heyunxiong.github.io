Hexo+GitHub博客搭建

Yunxiong

技术

发布于：Apr 17, 2019

即使没有自己买服务器去搭建， 可以借助GitHub Page免费托管功能，也可以实现静态博客的搭建！

安装Git

git的安装, 只要被配置好自己的账号即可！

安装Node.js

从官网下载，直接安装就好 https://nodejs.org/en/

查看是否看安装成功，会显示版本信息

安装Hexo

按照官网的步骤： https://hexo.io/zh-cn/

代码版本控制平台

  • 国内：Gitee码云，Coding

  • 国外：GitHub

    在GitHub创建对应的仓库，GitHub的必须要用 “用户名.github.io” 给仓库起名字。

    例如：yourname.github.io

推送到远程github仓库配置

在hexo的blog目录下找到 blog/_config.yml

购买域名(可选)

推荐购买域名服务提供商：

  • 国内：阿里旗下的万网
  • 国外 ：namesilo godaddy

关联购买的域名

  • 创建CNAME文件

    • 写上购买的域名

    • 在购买的域名服务商设置映射信息

      可以参考阿里云域名配置

详细搭建视频教程

https://www.bilibili.com/video/av44544186

from CodeSheep
